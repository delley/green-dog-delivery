# Green Dog Delivery

Aplicação web desenvolvida com Spring Boot utilizando as seguintes tecnologias:

* Spring MVC;
* Spring Data;
* Spring HATEOAS;
* Thymeleaf;
* Bootstrap;
* JQuery;

## Para executar o projeto

Baixe os fontes do projeto e execute:

```
mvn spring-boot:run
```


Em seguida acesse em `http://localhost:8080`
